# ConvictionalSeller::PatchVariantImages

## Properties

| Name | Type | Description | Notes |
| ---- | ---- | ----------- | ----- |

## Example

```ruby
require 'convictional_seller'

instance = ConvictionalSeller::PatchVariantImages.new(
```
